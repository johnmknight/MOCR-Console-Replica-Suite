# MOCR Reference Image Collection

Open source reference images for the Apollo-era Mission Operations Control Room (MOCR 2)
console stations and front-wall projection screens. Supports development of the
MOCR Console Replica Suite and serves as an educational resource for anyone studying
Mission Control history.

## Purpose

- Provide visual references for each console position's physical layout, displays, and controls
- Document the front-wall projection screens and chronographic displays
- Include both historical Apollo-era photographs and 2019 restoration images
- Supply annotated diagrams that highlight specific controls, labels, and display formats

## Licensing & Attribution

**All images in this collection must be freely usable.** Accepted sources:

- **NASA images** — Works created by NASA are in the **public domain** (not subject to
  U.S. copyright). Always credit: `NASA / [mission or center]`
- **U.S. Government works** — Similarly public domain. Credit the originating agency.
- **Creative Commons (CC BY, CC BY-SA, CC0)** — Acceptable with proper attribution.
- **Personal photographs** — If you took the photo yourself, you may contribute it
  under CC BY-SA 4.0 or release it to public domain.

**Do NOT include:**
- Copyrighted press photos (AP, Getty, Reuters, etc.)
- Images with unclear licensing
- Screenshots from copyrighted documentaries or films

When adding an image, **always** add an entry to the attribution table below.

## Naming Convention

All images go in this single folder. Use this naming pattern:

```
{station}-{subject}-{detail}.{ext}
```

**Station prefixes** (one per console position + screens):

| Prefix | Position |
|--------|----------|
| `booster` | Booster Systems Engineer |
| `retro` | Retrofire Officer |
| `fdo` | Flight Dynamics Officer |
| `guido` | Guidance Officer |
| `surgeon` | Flight Surgeon |
| `capcom` | Capsule Communicator |
| `eecom` | EECOM (Electrical, Environmental) |
| `gnc` | GNC (Guidance, Navigation, Control) |
| `telmu` | TELMU (Telemetry, Electrical, EVA Mobility Unit) |
| `control` | Control Officer |
| `inco` | Instrumentation & Communications Officer |
| `procedures` | Operations & Procedures Officer |
| `fao` | Flight Activities Officer |
| `network` | Network Controller |
| `flight` | Flight Director |
| `pao` | Public Affairs Officer |
| `screen` | Front-wall projection screens |
| `clock` | Chronographic clock displays |
| `mocr` | General room views / overviews |
| `batcave` | Rear projection room |

**Subject keywords:**

| Keyword | Meaning |
|---------|---------|
| `console` | Full console overview / wide shot |
| `display` | CRT or display panel close-up |
| `controls` | Buttons, switches, knobs detail |
| `panel` | Specific panel section |
| `diagram` | Annotated diagram or schematic |
| `label` | Label or nameplate close-up |
| `trajectory` | Trajectory plot on projection screen |
| `groundtrack` | Earth ground track display |
| `restoration` | 2019 restoration photo |
| `overview` | Wide shot showing multiple stations |

**Examples:**

```
fdo-console-overview.jpg          — Wide shot of the FDO console position
fdo-display-trajectory.png        — FDO CRT showing trajectory data
screen-groundtrack-apollo13.jpg   — Center screen during Apollo 13 Earth orbit
screen-trajectory-descent.jpg     — Powered descent trajectory plot
clock-get-closeup.jpg             — GET clock close-up
mocr-overview-apollo11.jpg        — Full room view during Apollo 11
booster-controls-detail.jpg       — Close-up of Booster console switches
eecom-diagram-annotated.png       — Annotated diagram of EECOM panel layout
flight-console-restoration.jpg    — Flight Director console, 2019 restoration
batcave-projectors-overview.jpg   — Rear projection room equipment
```

## Known Sources for Public Domain MOCR Images

- **NASA Image and Video Library** — https://images.nasa.gov
  Search: "Mission Control", "MOCR", "Mission Operations Control Room"
- **NASA JSC History Collection** — https://historycollection.jsc.nasa.gov
- **Project Apollo Archive (Flickr)** — https://www.flickr.com/photos/projectapolloarchive/
  Thousands of unprocessed Apollo mission photos, all public domain.
- **Ars Technica MOCR Restoration Series** — Extensive high-res photos from the 2019
  restoration (verify individual image licensing before use).
- **Smithsonian National Air and Space Museum** — Digital collection may have MOCR artifacts.
- **collectSPACE.com forums** — Community photos from JSC tours (check individual licensing).
